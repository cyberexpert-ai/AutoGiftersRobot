| Command | Action |
|---|---|
| `/admin` | Open full admin panel |
| `/stats` | Quick statistics |
| `/broadcast` | Start broadcast |
| `/ban USER_ID` | Ban a user |
| `/unban USER_ID` | Unban a user |

| `/maint` | Toggle maintenance |
| `/cancel` | Cancel current action |
